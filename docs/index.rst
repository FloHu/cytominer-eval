Welcome!
==========================================

Cytominer-eval is a package facilitating rapid evaluation of cell-based profiles.
Profiles are high dimensional readouts from high-throughput experiments representing the impact of perturbation on cells.
This python package implements common metrics used to evaluate profile quality and strength.
